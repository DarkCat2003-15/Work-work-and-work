# 💼 Work-work-and-work

Repositorio donde documento los proyectos, ejercicios y prácticas que desarrollo durante mi experiencia laboral y mi proceso de aprendizaje como desarrollador FullStack.

## 📖 Descripción

Este repositorio reúne diferentes actividades realizadas con el objetivo de fortalecer mis conocimientos en desarrollo de software, principalmente utilizando Java y Spring Boot.

Cada carpeta representa un proyecto independiente que aborda conceptos específicos como:

- Arquitectura por capas
- APIs REST
- Conexión a bases de datos
- CRUD
- JPA / Hibernate
- Spring Boot
- MySQL
- Buenas prácticas de programación
- More...

El propósito es mantener un historial de mi evolución como desarrollador y contar con un portafolio organizado de los ejercicios desarrollados.

---

## 📂 Proyectos

### 🏦 Cajero
Aplicación desarrollada para practicar la lógica de negocio y la gestión de operaciones relacionadas con un sistema de cajero.

**Temas trabajados:**

- Java
- Spring Boot
- Arquitectura por capas
- CRUD
- Persistencia de datos

---

### 📚 Biblioteca
Proyecto orientado a la administración de libros y préstamos mediante una API REST.

**Temas trabajados:**

- Spring Boot
- JPA
- MySQL
- Relaciones entre entidades
- Validaciones

---

### 🎓 Estudiantes
Proyecto para gestionar información de estudiantes y practicar operaciones CRUD utilizando Spring Boot.

**Temas trabajados:**

- API REST
- CRUD
- Arquitectura MVC
- Persistencia de datos

---

### 🔗 Conexión
Ejercicio enfocado en comprender la configuración y conexión entre Spring Boot y una base de datos relacional.

**Temas trabajados:**

- Configuración de DataSource
- MySQL
- Spring Data JPA

---

### ✅ TodoList API
API REST para la administración de tareas.

**Temas trabajados:**

- Spring Boot
- CRUD
- JPA
- Arquitectura por capas
- Manejo de entidades
